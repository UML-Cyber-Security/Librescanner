# Librescan
The official free and open-source vulnerability scanner of the UMass Lowell Cybersecurity Club.

![Alt text](https://github.com/UML-Cyber-Security/Librescanner/blob/main/librescan.png)

## About
LibreScan is a Free and Open Source vulnerability scanner under the GPL-3.0 license that incorporates vulnerability assessment and exploitation of web applications, port scanning, vulnerable & outdated software, web crawling and directory bruteforcing.

## Features

* LibreScan is written in python3 and allows for summary reports in pdf format for quick reading and distrobution. Encryption services such as GPG are incorporated for safe transmissions via SMTP to prevent data disclosure. To prevent IP blacklisting, proxy services are include via Socks4/5 or HTTP(S), it is recommended to use proxies that are owned by the user.

* FOSS: Free and Open Source allows for everyone to view the code and make changes to improve the functioning of the application.
